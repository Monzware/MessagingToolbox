MessagingToolbox

An Eclipse plugin project for messaging.

The pluging makes it possible to add messages to queues, and read messages from queues.
(Reading is not yet implemented)


21-01-2012 - Version 1.0.0.201201212215

- Initial submit of Update site

- JBoss 5.1.0 messager provider plugin added
- Amazone SQS messager provider plugin added
